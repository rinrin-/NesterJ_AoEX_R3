0.5更新
本模拟器是基于NesterJ 1.12开发的PSP用NES模拟器！
和原版相比，主要更新如下:

1.模拟器制式可以选择PAL和NTSC
2.模拟器界面语言可以选择中文或者英文
3.添加FAT ROM读取模式，该功能可以支持中文文件名的读取
4.添加新的作弊功能，方便修改游戏
5.ROM支持最大4M
6.增加异常处理，防止读取游戏死机
7.增加ROM删除功能（ROM选择界面按□即可）
8.增加自定义ROM修改数据库，支持ROM头自定义修复

 注意
	DQ4(1的汉化版)要读2次才能游戏
	三国志中原之霸者外星中文版要读2次才能游戏。

 部分吞食天地2中文版 需要选择PAL制式，然后重新读取ROM，才能正常游戏，否则会花屏
 街头霸王9人版， 请选择NTSC/AUTO制式，然后加载游戏，否则游戏会白屏10多秒才能进入游戏
 赤色要塞，      请选择PAL然后读取ROM，否则会白屏10多秒
 街头霸王4人版   选PAL会花屏，请选择NTSC/AUTO制式
 南晶游戏        请将频率设置为333,比如《古墓丽影》，否则会卡到像死机一样
